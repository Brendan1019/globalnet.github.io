The way the URL, or web address, is written indicates the type of network the browser will access. Some of the most common large codes are as follows:

  • FTP: The File Transfer Protocol is used to transfer documents between laptop computers that are linked to the Internet.
  • USENET: The http:// prefix allows access to Usenet newsgroups on specified data servers. A web browser will most of the time be able to view a newsgroup from an information server that is appropriate for its settings.
  • ELECTRONIC MAIL: Forthe internet browser to send an email with the mailto: prefix to a specified address, an SMTP server address must first be configured.
  • TELNET: The Telnet protocol is used to connect a computer to a remote host. Most web browsers will launch a helper program for telnet connections.
  • HTTP: The Hypertext Transfer Protocol is the most often used way for accessing web pages, images, and other sorts of information on the World Wide Web.
  • WAIS: "Wide Area Information Servers" is an abbreviation for "Wide Area Information Servers." Browsers no longer frequently have direct access to WAIS. It is instead a list issuer, which is frequently used by CGI programs.
  • LOCAL FILES: You can use a web browser to see and explore local files and directories on your computer, even if they have nothing to do with Internet services.
  • HTTPS: Encrypted To serve sites that require users to submit credit cards or other private information, HTTP frequently includes a Secure Sockets Layer (SSL).
  • GOPHER: Gopher is an online statistics service that replaces a service that the World Wide Web has superseded.
